## ✨ Дополнительно (UX детали)

- **Hover-анимации:** `150–200ms`
- **Active:** нажатие с затемнением на `5–10%`
- **Disabled:**
  - `opacity: 0.4–0.6`
  - `cursor: not-allowed`
